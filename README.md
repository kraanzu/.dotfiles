# Kraanzu's NixOS Dotfiles

> [!NOTE]
>
> **System Information:**
>
> - **OS:** NixOS
> - **Window Manager:** Qtile
> - **Shell:** Fish
> - **Terminal:** Wezterm
> - **Editor:** Neovim


## Setup

```bash
sudo nixos-rebuild switch --flake .#nzxt
home-manager switch -b backup --flake .#nzxt
```
